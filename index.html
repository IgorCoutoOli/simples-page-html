Apresentação do Curso

História da Web (Voltar)

A Web, também conhecida como WWW, foi criada em 1989 por Tim Berners-Lee no CERN, e é um sistema de documentos interconectados acessados via internet. Originalmente, a Web foi desenvolvida para facilitar o compartilhamento de informações entre cientistas. Em 1993, a Web tornou-se disponível publicamente e gratuitamente, o que impulsionou seu crescimento rápido com o desenvolvimento de navegadores como Mosaic e Netscape. Isso levou à criação de milhares de sites e à popularização da Web como a conhecemos hoje​.

O que são Clientes? (Voltar)

No universo da web, o termo "clientes" é fundamental para entender a interação entre diferentes partes de um sistema. Ele se refere, predominantemente, aos navegadores que os usuários utilizam para acessar a internet, mas também abrange o conceito de comunicação entre cliente e servidor, um pilar da arquitetura de aplicações web.

Os navegadores web, como Google Chrome, Mozilla Firefox e Safari, atuam como clientes que interpretam e renderizam conteúdo web. Quando um usuário acessa um site, o navegador é responsável por interpretar o código HTML, CSS e JavaScript que compõe a página. O HTML estrutura o conteúdo, o CSS o estiliza e o JavaScript adiciona interatividade. Essa interpretação é essencial para a experiência do usuário, pois transforma dados brutos em uma interface visual interativa.

Por exemplo, quando um usuário insere um URL em seu navegador, ele inicia uma série de eventos. O navegador envia uma requisição HTTP ao servidor que hospeda o site, solicitando os recursos necessários para exibir a página. Essa comunicação é baseada no protocolo HTTP (ou HTTPS, para conexões seguras), que define como as mensagens são formatadas e transmitidas.

Em resumo, os clientes no contexto de HTML e desenvolvimento web desempenham um papel vital na experiência do usuário e na comunicação entre front-end e back-end. Eles não apenas interpretam e apresentam dados, mas também são responsáveis por facilitar a interação do usuário com as aplicações web. Compreender essa dinâmica é fundamental para qualquer desenvolvedor que busca criar aplicações web eficazes e intuitivas.

O que são Servers? (Voltar)

O termo servidor refere-se a um sistema que entrega arquivos e recursos da web a clientes, tipicamente navegadores. Esses arquivos e recursos podem incluir documentos HTML, folhas de estilo CSS, scripts JavaScript, imagens e outros tipos de mídia. O servidor desempenha um papel fundamental na arquitetura da web, facilitando a comunicação entre o cliente (navegador) e os dados ou conteúdos armazenados.

No contexto do HTML, o servidor é um componente vital que permite a entrega de conteúdo e a interação com usuários através da web. Ele processa requisições, fornece arquivos HTML e recursos associados, e facilita a criação de experiências dinâmicas e interativas. Compreender o papel do servidor é crucial para desenvolvedores que desejam construir aplicações web eficazes e responsivas.

Linguagens de Programação (Voltar)

Linguagens de programação são conjuntos de regras e instruções que permitem aos programadores escrever códigos que um computador pode entender e executar. Essas linguagens são usadas para desenvolver software, scripts e outras aplicações que podem executar tarefas específicas, desde operações simples até sistemas complexos.

As linguagens de programação podem ser divididas em várias categorias, incluindo:

  • Linguagens de Baixo Nível: Estas são mais próximas do código de máquina e incluem Assembly. Elas oferecem maior controle sobre o hardware, mas são mais difíceis de usar.
  • Linguagens de Alto Nível: Estas são mais abstratas e fáceis de entender, como Python, Java e C#. Elas permitem que os programadores escrevam código que é mais legível e compreensível para os humanos.
  • Linguagens de Script: Estas são frequentemente usadas para automação de tarefas e desenvolvimento web. Exemplos incluem JavaScript e PHP.

Linguagens de Programação Usadas em Conjunto com HTML

HTML é a linguagem de marcação padrão para criar páginas da web. Embora o HTML seja fundamental para estruturar o conteúdo, outras linguagens de programação são frequentemente usadas em conjunto para adicionar interatividade, estilo e lógica às páginas da web. Aqui estão algumas das principais linguagens que podem ser usadas em conjunto com HTML:

  1. CSS
  2. JavaScript
  3. PHP
  4. Python
  5. Ruby
  6. TypeScript

Linguagens de programação desempenham um papel vital no desenvolvimento de software e aplicações web. HTML serve como a estrutura básica para a criação de páginas da web, mas para criar experiências interativas e dinâmicas, é essencial integrar outras linguagens, como CSS e JavaScript. O uso combinado dessas linguagens permite que os desenvolvedores criem aplicações web ricas, funcionais e atraentes, atendendo às necessidades e expectativas dos usuários modernos.

Ferramentas Utilizadas (Voltar)

Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft, amplamente utilizado por desenvolvedores devido à sua interface amigável e rica funcionalidade. Ele suporta várias linguagens de programação e possui um vasto ecossistema de extensões que facilitam o desenvolvimento. Entre as extensões úteis para HTML, podemos destacar:

  • HTML Snippets: Fornece atalhos e snippets que ajudam a acelerar a escrita de código HTML comum, economizando tempo ao evitar a digitação completa de tags.
  • Live Server: Permite visualizar as alterações em tempo real no navegador, recarregando a página automaticamente sempre que o código é salvo, o que melhora o fluxo de trabalho.
  • Prettier: Uma ferramenta de formatação de código que garante que o HTML seja formatado de forma consistente, melhorando a legibilidade e a organização do código.
  • Emmet: Uma poderosa ferramenta que permite escrever código HTML e CSS de forma mais rápida através de abreviações que são expandidas em tags completas.
  • IntelliSense for CSS class names: Fornece sugestões de nomes de classes CSS enquanto você digita, ajudando a evitar erros de digitação e a encontrar rapidamente classes existentes.

Inspetor de Elementos (Voltar)

O Inspetor de Elementos é uma ferramenta de desenvolvimento integrada nos navegadores modernos, como Chrome e Firefox. Essa ferramenta permite que os desenvolvedores visualizem e editem o código HTML e CSS de uma página da web em tempo real. Você pode usar o inspetor para:

  • Examinar a estrutura de elementos HTML.
  • Modificar o CSS aplicado a esses elementos e ver as alterações imediatamente no navegador.
  • Depurar problemas de layout e estilos ao observar como diferentes regras CSS afetam a aparência da página.
  • Visualizar e alterar atributos de elementos, como classes, IDs e estilos inline.

A estrutura básica do HTML é o esqueleto fundamental de qualquer documento HTML. Uma página HTML típica segue este formato:

        
        <!DOCTYPE HTML>
        <HTML lang="pt-BR">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Título da Página</title>
        </head>
        <body>
            <h1>Bem-vindo ao Meu Site!</h1>
            <p>Este é um parágrafo de exemplo.</p>
        </body>
        </HTML>

Nesse exemplo:

  • <!DOCTYPE HTML>: Declara o tipo de documento, informando ao navegador que este é um documento HTML5.
  • <HTML>: A tag raiz que envolve todo o conteúdo da página.
  • <head>: Contém metadados sobre o documento, como o conjunto de caracteres e o título que aparece na aba do navegador.
  • <body>: Onde o conteúdo visível da página é colocado, incluindo textos, imagens e links.

Falando sobre Tags (Voltar)

As Tags são os elementos fundamentais do HTML que definem a estrutura e o conteúdo de uma página. Elas são usadas para organizar o conteúdo de maneira semântica, cada uma com um propósito específico. Exemplos de tags comuns incluem:

  • <p>: Usada para criar parágrafos.
  • <a>: Usada para criar links.
  • <img>: Usada para inserir imagens.
  • <div>: Um contêiner genérico usado para agrupar conteúdo.
  • <span>: Usada para aplicar estilos em partes específicas de texto dentro de um parágrafo.

Atributos das Tags (Voltar)

Os Atributos das Tags fornecem informações adicionais sobre os elementos HTML. Eles seguem o formato nome="valor" e são usados para modificar o comportamento ou a aparência das tags. Exemplos incluem:

  • href: Usado na tag <a> para especificar o URL de destino do link.
  • src: Usado na tag <img> para especificar o caminho da imagem.
  • alt: Usado na tag <img> para fornecer uma descrição da imagem, importante para acessibilidade.
  • class: Usado para aplicar estilos CSS a um grupo de elementos.
  • id: Usado para identificar um elemento específico de forma única.

Textos (Voltar)

Os Textos em HTML são representados por várias tags que definem diferentes níveis de importância e estilo. Exemplos incluem:

  • <h1> a <h6>: Usadas para definir títulos e subtítulos, onde <h1> é o mais importante e <h6> o menos importante.
  • <p>: Usada para parágrafos de texto.
  • <em>: Usada para enfatizar texto, geralmente exibido em itálico.
  • <strong>: Usada para destacar texto, geralmente exibido em negrito.
  • <blockquote>: Usada para citar uma parte de texto de outra fonte, geralmente exibida com recuo.

Lista Ordenadas e Não Ordenadas (Voltar)

As Listas Ordenadas e Não Ordenadas são usadas para agrupar itens relacionados. A lista ordenada é numerada, enquanto a lista não ordenada é apenas marcada. Exemplos incluem:

        
        <ol>
            <li>Primeiro Item</li>
            <li>Segundo Item</li>
        </ol>

        <ul>
            <li>Item A</li>
            <li>Item B</li>
        </ul>

As listas são úteis para organizar informações de forma clara e concisa, melhorando a usabilidade e a leitura da página.

Links (Voltar)

Os Links são criados com a tag <a>, que permite a navegação entre páginas da web ou seções dentro de uma mesma página:

        <a target="_blank" href="https://www.dio.me">Visite o site da Dio.me</a>
